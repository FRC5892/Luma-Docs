## About the Luma P1

The Luma P1 is an affordable vision co-processor released for the **2026 FRC REBUILT season**.
It pairs a camera with a Raspberry Pi CM5 co-processor and runs
[PhotonVision](https://photonvision.org/), a free open source vision software for FRC.

Key highlights:

- **Affordable** — designed with FRC budget constraints in mind
- **PoE supported** — power and data over a single Ethernet cable
- **PhotonVision powered** — familiar software stack for teams already using PhotonVision
- **Raspberry Pi CM5** — capable co-processor with an active community

!!! tip "Already familiar with Limelights?"
    The Luma P1 is conceptually similar to running a Limelight with PhotonVision,
    with the added benefit of PoE support (removed in Limelight 4).
