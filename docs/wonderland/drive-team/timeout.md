# Timeout Triage Script

Use a fixed timeout triage structure to stabilize the robot and reduce compounding errors.

- Define a 30-second script: symptoms, likely causes, immediate safe fallback
- Separate must-fix-now items from can-survive-one-match items
- End every timeout with one clear go or no-go owner
