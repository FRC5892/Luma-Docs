# Wonderland: Programming

## What Programming Does

Programming develops control logic, autonomous behavior, and diagnostics that make robot behavior predictable and performant.

The goal is predictable robot behavior through tested code, clear fallback paths, and disciplined releases.

Core responsibilities include:

- Stable deploy and rollback workflows
- Logging and telemetry for rapid diagnosis
- Driver-centric software reliability
- Safe software changes during competition
- Excessive caffine consumption
