# Comp Battery Tracking

Use a repeatable process for every competition battery:

Use measured battery data and repeatable tracking to guide all competition battery decisions.

- Assign each battery a permanent ID and visible tag
- Log every match use, charge cycle, and resting voltage
- Retire batteries that sag under load or repeatedly fail post-match checks
- Keep one charger lane reserved for emergency turnarounds only
