# Loctite Application Checking

## UV-Reactive Bolt Coating for Threadlocker Detection

Use this method to verify threadlocker application and detect migration with objective inspection.

### Purpose

This procedure describes how to prepare fasteners with a UV-reactive surface coating that enables visual detection of threadlocker contamination using ultraviolet inspection light.

When threadlocker contacts the coated surface, a fluorescence contrast halo appears under UV illumination, indicating adhesive migration beyond the threaded interface.

### Materials Required

- Isopropyl alcohol (70% or greater)
- Lint-free wipes
- Matte white spray paint
- Clear UV-reactive spray coating (fluorescent clear coat)
- Disposable gloves
- Drop surface or spray tray
- UV inspection flashlight (365-395 nm recommended)

### Surface Preparation

1. Wear disposable gloves.
2. Clean bolt heads using isopropyl alcohol.
3. Remove oils, debris, and machining residue.
4. Allow bolts to fully dry before coating.

Proper surface preparation is required for reliable fluorescence response.

### Base Coat Application

1. Place bolts on a protected spray surface.
2. Apply a thin, even layer of matte white spray paint.
3. Avoid excessive paint buildup.
4. Allow the base coat to dry completely per manufacturer recommendations.

The matte white layer improves fluorescence visibility and contrast under UV inspection.

### UV-Reactive Coating Application

1. Apply a thin layer of clear UV-reactive spray coating.
2. Ensure full coverage of bolt heads.
3. Avoid pooling or dripping.
4. Allow coating to cure fully before installation.

This layer enables fluorescence contrast detection when threadlocker contacts the surface.

### Installation Procedure

1. Install coated fasteners normally.
2. Apply threadlocker only to threaded regions.
3. Avoid intentional coating of bolt heads unless detection marking is desired.

If threadlocker migrates beyond the threaded interface, it becomes detectable during inspection.

### Inspection Procedure

1. Illuminate fasteners using a UV flashlight (365-395 nm).
2. Observe bolt head surfaces for fluorescence contrast halos.
3. Presence of halo indicates threadlocker contact with coated surface.

Example inspection outcomes:

| Observation | Interpretation |
| --- | --- |
| Uniform fluorescence | No detected threadlocker contact |
| Localized bright halo | Threadlocker migration present |
| Irregular fluorescence patch | Possible contamination or overapplication |

### Advantages

- Low material cost
- Simple application workflow
- Non-destructive inspection
- Works with standard inspection UV tools
- Suitable for robotics maintenance workflows

### Limitations

- Detection requires UV illumination
- Excess paint thickness may reduce sensitivity
- Surface oils reduce coating effectiveness
- Some threadlocker formulations fluoresce differently than others

### Recommended Use Cases

This method is appropriate for:

- Robotics assemblies
- Maintenance inspection workflows
- Repeatable subsystem verification
- Training environments requiring visual confirmation of adhesive use

It is not intended as a substitute for torque verification or witness marking systems.
