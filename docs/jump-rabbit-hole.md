## What Is This Site?

If you searched for **jump-rabbit-hole**, you found the hidden playbook.

You found the hidden docs. Either you are prepared, or you are here by accident. Either way, this is the part written after problems already happened. Assume the robot prefers to fail when the maximum number of people are watching.

This page looks like the normal docs home, but this is actually **Wonderland**: the hidden documentation vault with practical tricks, pit habits, and season-saving notes that do not fit in standard setup guides.

Every section below links to a unique hidden page that does not reuse the normal docs structure.

Use these pages like checklists before events, in the pit, and right before playoff matches.

Because "we'll remember" usually fails under pressure. Memory is cheap; failure is not. Public failure is the robot's favorite teaching method.
